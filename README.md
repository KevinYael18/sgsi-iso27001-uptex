Plataforma SGSI ISO/IEC 27001:2022

Descripción

Sistema web para la administración de un Sistema de Gestión de Seguridad de la Información (SGSI) conforme a la norma ISO/IEC 27001:2022.

Objetivo

Centralizar la gestión de activos, riesgos, incidentes, auditorías y documentación institucional.

Tecnologías

* Laravel
* PHP
* MySQL
* Bootstrap
* Docker
* GitHub Actions

Arquitectura

Cliente-Servidor
